# Router 产品线术语与句库

## 1. 术语

### IPv4
- English: Internet Protocol version 4
- Chinese: 互联网协议版本 4
- Definition: Protocol for addressing and routing packets in networks.

### DHCP
- English: Dynamic Host Configuration Protocol
- Chinese: 动态主机配置协议
- Definition: Assigns IP addresses to devices automatically.

### NAT
- English: Network Address Translation
- Chinese: 网络地址转换
- Definition: Maps private IP addresses to a public IP for internet access.

---

## 2. 常用句子

### Router-FUNC-001
- English: The router supports DHCP server functionality.
- Chinese: 路由器支持 DHCP 服务器功能。
- Usage: Manual

### Router-FUNC-002
- English: NAT allows multiple devices to share a single public IP.
- Chinese: NAT 允许多台设备共享一个公网 IP。
- Usage: Manual
