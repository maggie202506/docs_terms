# Switch 产品线术语与句库

## 1. 术语

### VLAN
- English: Virtual Local Area Network
- Chinese: 虚拟局域网
- Definition: A method to create logically separate networks within the same physical network.

### PoE
- English: Power over Ethernet
- Chinese: 以太网供电
- Definition: Technology that allows network cables to carry electrical power.

### Port
- English: Port
- Chinese: 端口
- Definition: Physical or logical network interface on the switch.
